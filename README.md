# Ahorcado-ApiRest
 Juego del ahorcado en una API Rest con Spring

---

## _METHOD GET_

Get all active games
```
  http://localhost/game
```
Get one game
```
  http://localhost/game/{id}
```
Get logs
```
  http://localhost/game/logs
```

---

## _METHOD POST_

Create game
```
  http://localhost/game/{secretWord}
```
Send letter to one game
```
  http://localhost/game/{id}/{letter}
```
